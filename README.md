## Steps on How to Create a GitHub Page
1. Create a GitHub repository.
2. Add your HTML and CSS files to the repository.
3. Go to the repository **Settings**.
4. Navigate to **Pages**.
5. Select the branch containing your website files.
6. Save the settings and wait for GitHub to deploy the page.
7. Access the website using the generated GitHub Pages link.
